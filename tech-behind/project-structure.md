# Project Structure

#### Our project is 100% open-source.

#### Each and every commit is public since the moment we started the project.

#### Our repositories:

\[[Github](https://github.com/cape4labs/audatadao_proof_of_contribution)] - the script we use for evaluating value of user submitted data. \[[read about it here](proof-of-contribution.md)]

\[[Github](https://github.com/cape4labs/audatadao_data_refiner)] - the script we run against data considered valuable transforming it into sql queryable format.

\[[Github](https://github.com/cape4labs/audatadao_dlp_ui)] - code of the web Dapp where users submit their audio data.

\[[Github](https://github.com/cape4labs/audatadao_user_db_backend)] - our backend service we use to maintain the users DB, the leaderboard, etc.
